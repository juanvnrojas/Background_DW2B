background-image

O que é?

Propriedade CSS para definir imagens ou gradientes como fundo de elementos.

Como usar?

Sintaxe básica:
background-image: url("imagem.jpg");

Exemplo com imagem

Gradientes CSS

Podemos criar degradês usando:

Linear Gradient
Radial Gradient

Sintaxe de Gradientes:

/* Linear */
background-image: linear-gradient(direção, cor1, cor2);

/* Radial */
background-image: radial-gradient(forma tamanho, cor1, cor2);

Exemplo Prático:

Importante saber:

  • Gradientes não precisam de arquivos externos
  • Usar background-position para ajustar foco
