background-repeat

O que faz?

Controla como uma imagem de fundo se repete no elemento.

Valores principais:

background-repeat: repeat; /* Padrão */
background-repeat: no-repeat;
background-repeat: repeat-x;
background-repeat: repeat-y;
background-repeat: space;
background-repeat: round;

Exemplos práticos:

repeat (padrão)
no-repeat
repeat-x (horizontal)
repeat-y (vertical)

Importante saber:

  • Funciona junto com background-image
  • O background-size afeta o padrão de repetição
  • Usar no-repeat para imagens únicas
